# Polymarket-Kalshi_vs_OIS-cut-probabilities
Visualizes differences in pricing of cuts by FED between Fed-Fund-Futures, Overnight Index Swaps and Preditcion markets. 
Kalshi: one API call gets all meetings, prices in cents → divide by 100.
Polymarket: one API call per meeting using hardcoded event slugs, reads lastTradePrice directly.
